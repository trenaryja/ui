# Roadmap

Planned form input components for future implementation.

## Text Input Variants

- **NumberInput** - Input with +/- stepper buttons, min/max/step validation
- **PasswordInput** - Input with show/hide toggle
- **PinInput / OTPInput** - Individual boxes for verification codes

## Selection Components

- **Combobox / Autocomplete** - Select with type-to-filter functionality
- **MultiSelect** - Select multiple values, displayed as chips/tags
- **CheckboxGroup** - Like RadioGroup but for checkboxes (multi-select)

## Tag & Token Inputs

- **TagInput** - Free-form text that creates removable tags

## Date & Time

- **DatePicker** - Calendar-based date selection
- **TimePicker** - Time selection
- **DateRangePicker** - Select a start and end date

## File & Media

- **FileInput** - Styled file upload with drag-and-drop support

## Specialized Inputs

- **ColorInput** - support for oklch and oklab
